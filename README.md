# ❤️ Donate Life AI

A Streamlit-based web application for managing organ donation details including donors and patients with proper validation and clean UI.

---

## 🚀 Features

- User Login (Email / Phone validation)
- Donor Registration Form
- Patient Details Form
- Input validation (email, phone, name)
- Clean and responsive UI using Streamlit
- Simple data handling using Pandas

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
